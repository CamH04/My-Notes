LORAWAN

LoRaWAN (Long Range Wide Area Network), low power long range

sub-GHz frequency bands, such as 868 MHz in
Europe and 915 MHz in North America

Up to several KM's

pread-spectrum modulation technique called chirp spread spectrum (CSS), which allows it to transmit data over long distances while consuming very little power

encryption and
authentication

 

star-of-stars topology in
which gateways relay messages between end-devices and a central network server

gateways are connected to the network server via standard IP connections and act as a transparent bridge, simply converting RF packets to
IP packets and vice versa.

 

DEVICES:

• Class A devices are the most common and consume the least amount of
power. They have a simple communication scheme, where they can
transmit data to the gateway at any time, but can only receive data from
the gateway immediately after a transmission.
• Class B devices have an additional receive window, where they can
receive data from the gateway at specific times, providing improved
network performance.
• Class C devices have a continuously open receive window, allowing them
to receive data from the gateway at any time, but consume more power
than Class A and B devices.

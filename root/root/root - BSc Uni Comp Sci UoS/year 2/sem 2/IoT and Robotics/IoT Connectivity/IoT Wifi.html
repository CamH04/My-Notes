IoT Wifi

based on the IEEE 802.11 family of standards for WLANS

 

Stds:

• 802.11a: 5 GHz band, 54 Mbps data rate
• 802.11b: 2.4 GHz band, 11 Mbps data rate
• 802.11g: 2.4 GHz band, 54 Mbps data rate
• 802.11n: 2.4/5 GHz bands, 600 Mbps data rate
• 802.11ac: 5 GHz band, 1 Gbps data rate
• 802.11ax: 2.4/5 GHz bands, up to 10 Gbps data rate

 

Layers:

Physical : handles stuff like frequency, channel width, etc

Mac: manages the access to the wireless
medium, including transmission, reception, and collision avoidance

Network: handles routing

Transport layer: provides reliable data transfer services between applications running on different devices

 

 

Security:

• WPA2 and WPA3: Wi-Fi Protected Access protocols that use encryption and
authentication to secure the network.
• EAP (Extensible Authentication Protocol): a framework for authenticating
network clients using various methods, such as passwords, digital certificates,
and biometrics.
• 802.1X: a protocol that enables port-based authentication and authorization for
network access control.

 

 

Stds For IoT:

• Wi-Fi HaLow: a low-power, long-range Wi-Fi standard that operates in the sub-GHz bands and supports data rates of up to 347 Kbps, supports ipv6 , desgined for p2p or start topologys


• Wi-Fi 6E: an extended version of Wi-Fi 6 that operates in the 6 GHz band and provides higher data rates and lower latency for IoT applications.

Https Captures

  • Start a new Wireshark capture (no filters) on eth0
  • From the client machine access the www.honeypot.com site from the browser as follows:
    1.  
      1. http://www.honeypot.com
      2. https://www.honeypot.com

Stop the capture and filter the traffic as follows:

  1. Filter all the HTTP traffic
  2. Find the first HTTP request and follow the HTTP Stream and identify the HTML.
  3. Filter all the HTTPS traffic (not so straight forward a bit of research required by yourself!)

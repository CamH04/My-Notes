Merging Traffic

  • Start Wireshark and capture the traffic on eth0
  • Open a terminal on the VM Client and execute the command
    1.  
      • nslookup www.honeypot.com
  • Stop the capture on Wireshark and save the full capture to file “capture1.pcap”.
  • Restart the Wireshark capture on eth0
  • Open a terminal on the VM Client and execute the command
    1.  
      • nslookup www.sunderland.ac.uk
  • Stop the capture on Wireshark and save the full capture to file “capture2.pcap”.
  • Merge the two captures into a file named “combined.pcap”

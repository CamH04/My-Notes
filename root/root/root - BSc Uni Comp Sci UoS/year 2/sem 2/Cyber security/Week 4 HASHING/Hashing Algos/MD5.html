MD5

message-digest algorithm

 

Used often the verify the integrity of files
Not suitable to be used in cryptographic techniques

 

  • Data is broken into 512-bit blocks
  • Each block is broken down into 16 words of 32 bits each
  • A mathematical algorithm is applied to each block of data
  • The output from each block is combined with the output from the previous block
  • The final output from the last block is the MD5 hash

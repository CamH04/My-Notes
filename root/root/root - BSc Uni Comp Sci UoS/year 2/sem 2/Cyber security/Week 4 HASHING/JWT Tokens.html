JWT Tokens

Json web tokens

 

JWTs are compact, self-contained tokens that are URL-safe 

  • They are made up of three parts: a header, payload, and signature 
  • The token is a long string that's base64 URL-encoded 
  • JWTs can be sent through a URL, POST parameter, or HTTP header 
  • The recipient doesn't need to call a server to validate the token
